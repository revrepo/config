How to Modify a Domain

Use the following PUT to modify the domain. Again, using the object_id from the response header from when the domain was created.

https://api.revapm.net/v1/domain_configs/

cURL example:

curl -v -H "Accept: application/json" -H "Authorization:X-API-KEY 6905f1bd-cbab-4d82-ad9d-f818d1fbdad2" -X PUT -T Domain.json https://api.revapm.net/v1/domain_configs/581241b8eba90b004d3a6101

 

Portal Shows the change from origin.sample.com to origin-new.example.com.
