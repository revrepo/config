How to Purge Cached Objects

The object purging functionality is available for RevAPM Admins only.

To flush an object or set of objects from the RevAPM edge cache, do the following:

  1. Identify the web domain and the set of objects to be purged.

For example, it can be a single object  like “/favicon.ico”, and a set of objects identified by a regular expression like “/images/*.img”.

  1. On the Web menu, click Purge Cached Objects in the left panel.
  2. In the Domain dialog box, select the domain from the drop down menu.
  3. Define purging instructions.

For example, purge all PNG files under /images, non-recursive. Files under /images/today/ will not be purged.
