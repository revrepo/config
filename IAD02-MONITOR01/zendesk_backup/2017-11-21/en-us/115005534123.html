How to migrate a DNS domain to nuu:bit

You can migrate a DNS domain to nuu:bit by:

  1. Select DNS Service -> DNS Zones, and then click the orange Add New DNS Zone button.

 

  2. Enter a fully qualified domain name (e.g. domain.com), and then click the orange Create DNS Zone button. Please note that you may enter a subdomain like "sub.domain.com" but you need to control the DNS for the first-level domain.

 

  3. Click the Manage Records icon, and then edit the content of the DNS Zone. Please note that if you click the Auto-Discover Records button, our system will attempt to automatically discover the content of the DNS zone by probing all well-known and popular host names typically found in a DNS zone. The Auto-Discover Records tool does not guarantee that the discovered zone content will be absolutely complete or correct so you will need to manually review the proposed DNS zone content and approve it for publishing.

 

  4. Configure your DNS name registrar to use the name servers provided by our system for the DNS zone.

  5. Confirm that the domain's NS records are pointing to the expected name servers by clicking the Check DNS Zone Integration Configuration icon.
