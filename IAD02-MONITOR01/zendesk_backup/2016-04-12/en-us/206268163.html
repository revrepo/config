How to Delete a User

To delete a user, send a DELETE request.

https://api.revapm.net/v1/users/55bff4b8c01a42a2348170f7

cURL example:

curl  -u 'user@example.com:password' -X DELETE -H 'Content-Type: application/json' https://api.revapm.net/v1/users/55bff4b8c01a42a2348170f7

Response body:

{
  "statusCode": 200,
  "message": "Successfully deleted the user"
}
